## local development

init,

```bash
$ cat <<EOF > .env
NOTION_API_SECRET=<YOUR_NOTION_API_SECRET>
DATABASE_ID=<YOUR_DATABASE_ID>
EOF

# install
$ yarn install
```

dev,

```bash
$ yarn dev
```
